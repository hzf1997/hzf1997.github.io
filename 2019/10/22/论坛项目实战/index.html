论坛项目实战一

发表于 2019-10-22 | 分类于 项目

##第一篇
#快速下载项目并打开项目。

  1. git clone https://github.com/codedrinker/community 从git上克隆项目
  2. cd community 进入项目
  3. mvn flyway:migrate -Pdev 执行sql脚本
  4. mvn clean compile package 打包
  5. java -jar java -jar target/community-0.0.1-SNAPSHOT.jar
  6. 浏览器中打开localhost:8887

#导入IDEA

  1. 选择import project
  2. 选择community
  3. next
  4. 导入成功后打开CommunityApplication类运行即可
